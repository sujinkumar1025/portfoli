Medbot

It's a medical chatbot for consulting patients in an hospital

Medbot

Heavy usage of chat platforms by users, allied with developments in natural offer a favourable scenario for organizations to implement use cases This research identifies factors that influence the suitability of use cases user interfaces, enabling organizations to make more informed medbots implementations. The factors identified are grouped in three factors, factors to be considered to implement a chatbot over a human operator; that should be considered when implementing a chatbot over a traditional A use case selection process is created that uses the factors gathered and the more appropriate use cases between a set of potential use cases for medbot The selection process is applied to several use cases in the health care yielding the use case of scheduling a medical appointment with highest priority. A architecture of a medbot for scheduling an appointment is defined and User tests are conducted, comparing the interactions with the medbot with a graphical user interface. User tests indicate the chatbot as more efficient than the and most testers indicate it as a preferable method for scheduling an appointment, comparing to the website.
